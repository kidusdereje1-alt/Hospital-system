Hospital Management System
Purpose

The purpose of this project is to develop a simple Hospital Management System using Java while demonstrating important Object-Oriented Programming (OOP) concepts. The system helps manage basic hospital information such as patients and doctors.

This project was created as an academic assignment to practice Java programming and understand how OOP principles work in real applications.

Functions of the System

The Hospital Management System performs the following functions:

Stores patient information
Stores doctor information
Accepts user input using the Scanner class
Displays hospital details on the console
Handles invalid input using exception handling
Organizes data using ArrayList
Demonstrates inheritance and polymorphism through different classes
OOP Concepts Implemented
Encapsulation

Private variables are used with getters and setters to protect data.

Constructors

Constructors are used to initialize patient and doctor objects.

Inheritance

The Patient and Doctor classes inherit from the Person class using extends.

Method Overriding

The displayInfo() method is overridden in child classes.

Polymorphism

Objects of different classes are stored in an ArrayList<Person>.

Abstraction

An abstract class Person is used as the base class.

Packages

The program uses the hospital package for organization.

Access Modifiers

The project uses private, protected, and public access modifiers.

Exception Handling

try-catch is used to handle invalid user input.
